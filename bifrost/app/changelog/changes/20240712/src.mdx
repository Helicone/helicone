### What's new? 

- Sessions is now available! Trace and debug your agent workflows easily, did we mention we have tree/spans/chat views? 
- Helicone now has mem0 and Lytix integration
- Helicone is featured on Upstash’s blog! Here’s how we handled billions of LLM logs with Upstash Kafka and Cloudflare Workers. 
- We received an overwhelming number of projects using Helicone, check them out! 


### Changelog

- Improved filters (now 2x faster). On Requests page only. 
- Fine-tuned models now appear on Playground. 
- Implemented Async Logging Package using OpenLLMetry. 
- Added Replicate support. 
- Added sort on card view. 
- Input modal can now be viewed in Markdown. 
- Added doc on how to enable stream_usage header. 
- Added timeout for cached responses. 


### Bug Fixes

- Solved rate limiting bug. 
- Solved bug related to prompt deletion and creation.  
- Fixed sticky column sorting bug on refresh.
