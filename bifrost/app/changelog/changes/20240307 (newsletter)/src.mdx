### Feature updates

#### Prompts (Beta) 
You can now manage and view prompts within Helicone. While this feature is currently in beta, you will soon be able to...
- Experiment with prompts
- Run regression tests
- Evaluate different prompts

Monitor my prompts. 


#### On-Prem improvements

Helicone’s on-prem offering has improved, enabling quick setup with just a few commands. We've dedicated significant time to ensuring Helicone’s compatibility with Kubernetes.


Contact us to deploy on-prem. 



### Bitesize updates 🍪

🍪 Secure your application with 1 line of code. Simply add the 'Helicone-Prompt-Security-Enabled' header to your LLM requests for automatic detection and blocking of calls that fail security checks.

🍪 Claude-3 Support. Added support for Anthropic’s latest model, we now fully support streaming and price calculations.

🍪 Time to first token. Helicone now offers the ability to track time to the first token, so you can gain transparency into timing differences across model providers and platforms. 

🍪 Receive 6 months free of Helicone Pro when you refer a friend or company. Just go to “Dashboard” > “Refer a friend”.
