### Feature updates

#### Prompts (beta) 

[image]

You can now <a href="https://docs.helicone.ai/features/prompts" target="_blank">manage and version prompts</a> within Helicone. While this feature is currently in beta, you will soon be able to...

- Experiment with prompts
- Run regression tests
- Evaluate different prompts


#### On-prem improvements

[image]


Helicone's on-prem offering has improved, enabling quick setup with just a few commands. We've dedicated significant time to ensuring Helicone's compatibility with Kubernetes. <a href="https://www.helicone.ai/contact" target="_blank">Contact us to deploy on-prem</a>. 


### Bitesize updates

🍪 **Secure your LLM app with 1 line of code.** Simply add the `Helicone-LLM-Security-Enabled` header to your LLM requests to automatically detect and block calls that fail security checks. <a href="https://docs.helicone.ai/features/advanced-usage/llm-security" target="_blank">Here's our doc</a>. 

🍪 **Claude-3 Support.** Added support for Anthropic's latest model, we now fully support streaming and price calculations.

🍪 **Time to first token.** Helicone now offers the ability to track time to the first token, so you can gain transparency into timing differences across model providers and platforms.
