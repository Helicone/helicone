Docker images on Docker Hub
From now on, we will publish Docker images on Docker Hub and the GitHub Container Registry.

This change made deploying Langfuse easier across various platforms that do not natively support the GitHub Container Registry (e.g., Google Cloud Run). For more details, see the updated self-hosting instructions.
