### What's new? 

- GPT-4o cost support is now available! 
- Implemented Kafka! This means Helicone has persisted logs and is now 100x more scalable. Helicone will not lose your logs even if our service goes down, and can handle huge volume of requests.  
- Introduced Experiments! You can create versions of your prompt and compare outputs, here’s the guide.
- Added Anthropic support to Playground. 
- Increased performance by 500% when filtering by custom property on the Request page. ⚡️
- Added filter by custom property on dashboard. 
- Revamped docs so you can find resources easily. Just starting out in Helicone? We created a section for you! 


### Changelog
