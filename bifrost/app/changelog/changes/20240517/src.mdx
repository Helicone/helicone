### What's new? 

- **GPT-4o cost support is now available!** 
- **Implemented Kafka!** This means Helicone has persisted logs and is now **100x more scalable**. Helicone will not lose your logs even if our service goes down, and can handle huge volume of requests.  
- **Introduced Experiments!** You can create versions of your prompt and compare outputs. 
- Added Anthropic support to Playground. 
- Increased performance by 500% when filtering by custom property on the Request page. ⚡️
- Added filter by custom property on dashboard. 
- Revamped docs so you can find resources easily. We added a section for you, <a href="https://docs.helicone.ai/getting-started/quick-start#new-to-helicone" target="_blank">if you're not sure where to start</a> in Helicone!

### Changelog

- Added an Openrouter integration. 
- We wrote a <a href="https://www.helicone.ai/blog/switch-models-safely" target="_blank">step-by-step guide on how to switch to gpt-4o safely with Helicone</a>. 
- Improve the UX for onboarding. 
- Officially deprecated Jobs. 
- Added a prompt mode Helicone header. 
- Added scores to Experiments to aggregate metrics between the production and the experiment prompt. 
- Established prompt over time metrics endpoint in Experiment. 
- Add deletes to Prompt page. 
- Add diff view for Prompt messages in Experiment. 
- Added filters to the Experiment log table.
