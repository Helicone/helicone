We're excited to announce a significant enhancement to our Python SDK with the introduction of `HeliconeLogBuilder`! This new feature provides a more robust and developer-friendly way to handle streaming LLM responses and logging in your Python applications.

## What's New

- **HeliconeLogBuilder Class**: A new builder pattern for simplified logging of LLM responses
- **Improved Streaming Support**: Better handling of streaming responses with automatic timing calculations
- **Enhanced Error Handling**: Built-in error tracking and formatting
- **Async Support**: Native support for asynchronous operations
- **Flexible Response Handling**: Support for both streaming and non-streaming responses

## Try It Now

Update your `helicone_helpers` package to version 1.0.3 to start using the new `HeliconeLogBuilder`:

```bash
pip install --upgrade helicone_helpers
```

Visit our <a href="https://docs.helicone.ai/getting-started/integration-method/manual-logger-typescript#heliconelogbuilder" rel="noopener" target="_blank">documentation</a> for detailed setup instructions and examples.
