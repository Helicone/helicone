### What's new? 

- **Introduced image logging.** Currently supporting Claude Vision, GPT-4 Vision, and DALL·E 3.
- **Moved to volumetric pricing.** Customers now only pay what they use, instead of monthly fixed. 
- Levelled up S3 support and removed payload size limitations. 
- Introduced Moderation Integration to safeguard chat completions. 
- Updated landing page and designed a better blog. 


### Changelog

- Moved our customer facing API from GraphQL to REST. <a href="https://docs.helicone.ai/rest/" target="_blank">REST API docs here</a>.
- We now have Google Gemini's Vertex AI Node.js SDK support. Here's <a href="https://docs.helicone.ai/integrations/gemini/vertex/javascript" target="_blank">how to integrate Helicone with Vertex AI</a>. 
- Improved Playground with a better UI and added image support 
- Added delete to dashboard persistent filters
- Added image support for prompts 
- Added sorting to prompts
- Set up endpoints to run experiments
- Updated our contributing guidelines on Github
- Added a request-by-country graph on dashboard
