## Quick Start

Most integrations follow a similar pattern:

1. **Get your Helicone API key** - Sign up at [helicone.ai](https://www.helicone.ai) and generate an [API key](https://us.helicone.ai/settings/api-keys)
2. **Update your configuration** - Change your base URL to `https://ai-gateway.helicone.ai/v1` and use your Helicone API key
3. **Start making requests** - Your existing code continues to work, now with full observability
